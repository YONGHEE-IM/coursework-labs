데이터 

컴퓨터 = 계산기 0, 1만으로 이루어졌다.
0 : 전기 안통함
1 : 전기 통함

트랜지스터 : 반도체 = 0, 1 = 전기를 보낼까 말까.

- 전기는 어디서 왔죠?
자석 돌리면 전기
금속에 열을 주면 전기

-> 전기를 주면 자석이 돌고 =  반도체
-> 전기를 주면 금속에 열 = 전열기 (헤어드라이기)

데이터 저장 원리
하드드라이브 : 물리적으로 홈을 판다. 핀셋으로 0, 1구분
CD : 레이저를 사용해서 0, 1을 구분한다.
usb : 반도체가 0, 1구분

- dbms 구성
1. 자연정보를 데이터화 : 추상화, 명확화
2. 데이터베이스 효율적 구성 : 정규형, 정규화, 인덱스
3. 데이터베이스 관리 : 추가, 수정, 삭제, 읽기/ 속도를 빨리/ 문법
